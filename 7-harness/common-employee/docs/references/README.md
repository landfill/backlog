# References

## 목적

정제 대상이 아닌 참고 자료를 보관한다.
이 폴더의 자료는 runbook이나 past-solution으로 직접 간주하지 않는다.

## 예시

- 아키텍처 설명 자료
- 정책 문서 사본
- 조사 배경 문서

## 규칙

- 운영 절차나 해결 사례로 승격할 자료는 `docs/domain-knowledge/raw/`에서 정제 파이프라인을 탄다
- `references/` 자료는 참조용으로만 보관하고, 티켓 처리 시 출처로 인용할 수 있다
- 민감 정보가 포함된 자료는 `docs/SECURITY.md` 기준을 그대로 따른다
