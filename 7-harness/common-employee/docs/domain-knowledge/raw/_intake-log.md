# 원본 자료 수집 로그 (Intake Log)

원본 자료를 수집할 때마다 아래 형식으로 기록한다.

## 로그 형식

```markdown
### YYYY-MM-DD — [자료명]
- **출처**: (Confluence URL / 파일 경로 / 제공자)
- **형태**: (위키 / PPT / Word / PDF / 기타)
- **수집 방법**: (API 추출 / 사용자 제공 / 직접 수집)
- **정제 상태**: (미정제 / 정제중 / 정제완료 / 검수완료)
- **정제 결과 위치**: (runbooks/... 또는 past-solutions/...)
- **비고**: (특이사항)
```
