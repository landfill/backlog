# Repository Claude Guide

이 문서는 Claude Code용 루트 진입 포인터다.

이 저장소의 공식 작업 진입 규칙과 read order의 단일 기준은 `AGENTS.md`다.
Claude Code는 루트 `AGENTS.md`를 먼저 읽고 그 규칙을 그대로 따른다.

- 공식 진입 문서: `AGENTS.md`
- repository-level harness와 app dispatch 규칙: `AGENTS.md`
- 대상 앱 진입과 상태 문서 확인 순서: `AGENTS.md`의 지시를 따른다

이 문서는 별도의 작업 규칙을 중복 정의하지 않는다.
