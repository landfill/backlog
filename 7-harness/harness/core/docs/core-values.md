# Core Values

## 1. Transparency
모든 판단에는 근거가 있어야 한다.
근거가 없는 행동은 하지 않고, 기록되지 않은 처리는 처리되지 않은 것과 같다.

## 2. Verifiability
문서, 구현, 테스트는 같은 동작을 가리켜야 한다.
검증할 수 없는 요구사항은 완료 기준으로 두지 않는다.

## 3. Safety
인증 정보와 사용자 데이터는 기본적으로 보호 대상이다.
명시된 근거 없이는 저장하거나 외부로 전송하지 않는다.

## 4. Simplicity
복잡하면 쪼갠다.
에이전트와 사람 모두가 읽고 따라갈 수 없는 구조는 줄인다.
