## 1. 개요

### 1.1 목적
손 제스처(스와이프, 핀치 줌)로 조작하는 웹 기반 프레젠테이션.
AI교육시나리오.md의 8개 섹션을 슬라이드로 구성하며, 강의 중 키보드/마우스 없이 진행 가능.

### 1.2 타겟 사용자
- **주 사용자**: 강사 (라이브 강의에서 직접 사용)
- **2차 사용자**: 교육 참석자 (슬라이드 내용 수신)

### 1.3 성공 기준
- [ ] 손 제스처로 슬라이드 전환이 신뢰성 있게 작동 (오인식률 < 10%)
- [ ] 핀치 줌 동작 (양손 핀치 인/아웃)
- [ ] 웹캠 없거나 손 인식 실패 시 키보드/마우스로 정상 작동
- [ ] 브라우저에서 로컬 실행 (서버 불필요)

### 5.2 제스처 인식 로직

**스와이프 감지 (단일 손)**
```
- 매 프레임: 손목(wrist) X 좌표 기록
- 최근 10프레임 이동량 계산
- |deltaX| > 화면 너비 15% + 속도 > 임계값
  - 오른쪽→왼쪽: Reveal.next()
  - 왼쪽→오른쪽: Reveal.prev()
- 쿨다운 800ms (연속 트리거 방지)
```

**핀치 줌 감지 (양손 동시)**
```
- 좌손 엄지끝(4) ↔ 검지끝(8) 거리 L
- 우손 엄지끝(4) ↔ 검지끝(8) 거리 R
- 두 손 모두 핀치 상태 (L < 40px AND R < 40px)
  - 이전 프레임 대비 양손 중심 거리 변화
    - 멀어짐 (+20px) → CSS scale UP
    - 가까워짐 (-20px) → CSS scale DOWN
- 핀치 해제 시 현재 줌 레벨 고정
```
